# NGワードチェッカー 実装計画書

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 設定単語（デフォルト + ユーザー追加）で入力テキストを部分一致＋正規化でチェックし、ヒット一覧と本文ハイライトで警告する、GitHub Pages 単一HTML のNGワードチェッカーを構築する。

**Architecture:** `index.html` 1ファイル完結（Tailwind CDN + 純JS・フレームワークなし）。JSは責務ごとに純粋関数として分離（normalize / checkText / localStorage I/O / 描画）。ハイライトは正規化の非1対1性を回避するため「元テキスト走査方式」を採用。ユーザー追加単語は localStorage に永続化。

**Tech Stack:** HTML5 + Tailwind CSS（CDN: `https://cdn.tailwindcss.com`）+ 純 JavaScript（ES2015+・フレームワークなし・ビルド不要）。ホスティング: GitHub Pages。

## Global Constraints

- 構成は**単一 HTML ファイル**（`index.html`）。JS を別ファイルに分離しない（Tailwind CDN も含めて1ファイル完結）。
- **コミットメッセージは日本語**・接頭辞付き（`追加:` `修正:` `整理:` `計画:` `文書:`）。プロジェクトルール準拠。
- 機種依存文字・中国語表現は使わない。自然な日本語・正書法を守る。
- **XSS 対策必須**: ユーザー入力（テキスト・単語）を HTML に挿入する箇所はすべて `escapeHtml` でエスケープ（`< > & " '`）。
- **テストは `test.html` をブラウザで開いて `console.assert` + 画面集計で検証**（Node 不要・設計準拠）。各テストケース前に `localStorage.clear()`。
- **完了報告前には必ず `test.html` を開いて全件合格を確認**し、結果（合格件数）を提示する。
- 単語の上限: 1単語100文字以下・ユーザー単語全体1000件以下。
- Tailwind CDN が不通でもロジックは動くよう、レイアウト（class）とロジック（JS）を分離して実装する。

```js
function normalize(str) {
  // 1. 小文字化
  let s = String(str).toLowerCase();
  // 2. NFKC 正規化（全角→半角・濁点結合など）
  s = s.normalize('NFKC');
  // 3. ひらがな(U+3041..U+3096) → カタカナ(+0x60)
  s = s.replace(/[ぁ-ゖ]/g, ch => String.fromCharCode(ch.charCodeAt(0) + 0x60));
  // 4. NFKC後も小文字化を再適用（半角化された英字の大小区別を統一）
  s = s.toLowerCase();
  return s;
}
```

```js
function checkText(text, words) {
  const normalizedText = normalize(text);
  const matches = [];
  for (const word of words) {
    const nw = normalize(word);
    if (nw === '') continue;  // 空単語は無視
    // 非重複スキップ方式で出現回数をカウント
    let count = 0;
    let pos = 0;
    while (true) {
      const found = normalizedText.indexOf(nw, pos);
      if (found === -1) break;
      count++;
      pos = found + nw.length;
    }
    if (count > 0) {
      matches.push({ word: word, normalizedWord: nw, count: count });
    }
  }
  return matches;
}
```

```js
// 本文をハイライト付きで描画（元テキスト走査方式・XSSエスケープ付き）
function renderHighlight(text, words) {
  const area = document.getElementById('highlight-area');
  if (text.trim() === '') {
    area.innerHTML = '<span class="text-gray-400">（チェック結果がここに表示されます）</span>';
    return;
  }
  if (words.length === 0) {
    area.textContent = text;  // textContent で安全に表示
    return;
  }

  // 正規化済み単語を準備
  const normalizedWords = words.map(normalize).filter(w => w !== '');

  // 元テキストを走査し、ヒット範囲 [i, j) を収集
  const ranges = [];
  let i = 0;
  while (i < text.length) {
    let matchedLen = 0;
    // 各単語について、text[i..] を正規化した先頭が一致するか確認
    for (const nw of normalizedWords) {
      // text[i..] を徐々に伸ばして正規化し、nw で始まる最小長を探す
      let acc = '';
      for (let len = 1; i + len <= text.length; len++) {
        acc = normalize(text.slice(i, i + len));
        if (acc.length < nw.length) continue;       // まだ短い
        if (acc.startsWith(nw)) {
          // nw の文字数分を消費する元テキスト長 = len
          if (len > matchedLen) matchedLen = len;
          break;
        } else {
          // acc が nw より長いのに先頭一致しない → この単語では不一致
          if (acc.length > nw.length) break;
        }
      }
    }
    if (matchedLen > 0) {
      ranges.push([i, i + matchedLen]);
      i += matchedLen;  // 非重複スキップ
    } else {
      i++;
    }
  }

  // エスケープしつつ範囲に <mark> を挿入
  let html = '';
  let cur = 0;
  for (const [start, end] of ranges) {
    html += escapeHtml(text.slice(cur, start));
    html += '<mark class="bg-yellow-300 rounded px-0.5">' + escapeHtml(text.slice(start, end)) + '</mark>';
    cur = end;
  }
  html += escapeHtml(text.slice(cur));
  area.innerHTML = html;
}
```
