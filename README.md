# NGワードチェッカー

テキストの中に「気をつけたい単語」が含まれていないか、かんたんにチェックできるツールです。

## 使い方

1. 公開されたURLにブラウザでアクセスします
2. 「チェック対象テキスト」の入力欄に、確認したい文章を貼り付けます
3. 「チェック」ボタンを押します
4. 右側に結果が表示されます:
   - **チェック結果**: ヒットした単語と回数
   - **ハイライト表示**: 本文の中で該当箇所が黄色く塗られます

## チェックする単語について

- **デフォルト単語**: あらかじめ設定された単語です（変更できません）
- **あなたの追加単語**: 自分で単語を追加できます。「追加する単語を入力」欄に入れて「追加」ボタンを押してください。追加した単語はあなたのパソコン/ブラウザに保存され、次回開いた時も残ります

※ 大文字/小文字、全角/半角、ひらがな/カタカナの違いは自動的に区別せずにチェックします（例: 「キーワード」と「きーわーど」は同じものとして扱います）

## 動作環境

モダンなブラウザ（Chrome / Edge / Firefox / Safari の最新版）で動作します。

## 公開について（GitHub Pages）

このツールは GitHub Pages で公開しています。公開ファイルは `index.html` と、設定単語をまとめた `default-words.js` です（リポジトリのルートにあります）。

公開設定（初回のみ・リポジトリ管理者が実施）:
1. GitHub のリポジトリページで「Settings」→「Pages」を開く
2. Source で「main ブランチ / ルート」を選択する
3. 設定後、数分で `https://<アカウント名>.github.io/<リポジトリ名>/` でアクセス可能になる

※ 公開ファイルを更新したら、`main` ブランチへ反映（push）するだけで公開サイトも自動更新されます。
